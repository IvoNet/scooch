# Live HTML

It is possible to add a Live HTML editor with this custom element:

```html
<live-preview-section>
    <textarea>
        <html>... some HTML ... </html>
    </textarea>
</live-preview-section>
```

---

<live-preview-section>
          <textarea>
<html>
  <head>
    <title>YOUR NAME</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
          </textarea>
        </live-preview-section>

# Smart Image

⚠️Highly Experimental⚠️

Make sure to start the server with:

```bash
OPENAI_API_KEY=<your OpenAI API key> npm start
```

And add this to your slides:

```html
<smart-image
  prompt="a unicorn with rainbow manes in a forest"
  width="700px" />
```
